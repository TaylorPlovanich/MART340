Gallery Layout

The gallery layout is inspired by museum displays, with artworks grouped into three main sections: Interactive Works, Video Works, and Static Works. Each section is organized to encourage exploration, with podiums showcasing individual pieces. Interactive works are positioned near the start to engage viewers immediately, while video and static works are placed deeper into the layout to invite further discovery. This flow mirrors the progression of a real-world gallery experience.

Navigation Features

The navigation system ensures a seamless experience. Visitors can use the top navigation bar to jump between sections or click on individual artworks to access detailed pages. Smooth scrolling and a back-to-top button enhance usability, making it easy to explore the exhibition.

Interactive Elements

Interactive features include clickable podiums that open detailed modal popups for each piece. Hover effects highlight active elements, inviting visitors to explore further. These features replicate the immersive nature of a physical exhibition, enhancing engagement within the digital space.
